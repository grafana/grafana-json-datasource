Configuration

Add a JSON data source#

  1. In the side menu, click the Configuration tab (cog icon)
  2. Click Add data source in the top-right corner of the Data Sources tab
  3. Enter "JSON" in the search box to find the JSON API data source
  4. Click the search result that says "JSON API"

The data source has been added, but it needs some more configuration before you can use it.
